# example.com

An about me web site.

## Dependencies

* PHP 7.2 or higher
* Composer
* An API key for Mailgun

## Installation

Clone this project into a web directory
```sh
cd /var/www
git clone git@github.com:microtrain1218/example.com.git
```

Install composer dependencies
```sh
cd /var/www/example.com
composer install
```
